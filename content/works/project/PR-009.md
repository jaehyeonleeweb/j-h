Document to Document for a Book is a web editor that displays the result of Markdown syntax with formatting applied as the user types. This editor was developed to support [Markdown to Book](/works/project/pr-008/), an automation technique that converts manuscripts written in Markdown into InDesign booklet data.
<br><br>
[Markdown to Book](/works/project/pr-008/) reads .txt and .md files and connects Markdown syntax to InDesign paragraph styles through a specific algorithm. In this process, the script can be executed regardless of which editor the user uses, as long as the manuscript exists as a text file (.txt or .md).
<br><br>
By contrast, Document to Document for a Book shows in real time how the entered Markdown syntax will be converted into formatted text, helping users who are not familiar with Markdown. If a user writes a manuscript using this editor, it can then pass through the automation process of [Markdown to Book](/works/project/pr-008/), allowing the user to proceed all the way to booklet production.
